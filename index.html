🔥 CONFIGURAR FIREBASE

Para usar o app, você precisa configurar seu projeto Firebase. Siga os passos abaixo e edite o arquivo index.html.

1. Acesse console.firebase.google.com e crie um projeto
2. Ative o Firestore Database no modo de produção
3. Em Configurações do projeto → Seus apps, clique em Web (</>) e copie o firebaseConfig
4. Cole os valores na variável FIREBASE_CONFIG no início do script deste arquivo
5. Nas Regras do Firestore, permita leitura e escrita temporariamente:
allow read, write: if true;
6. Faça o deploy no GitHub Pages e acesse o link gerado
